## 技术细节

### 错误解析
- 使用 `mypy --strict --no-pretty` 获取单行格式输出
- 正则表达式匹配: `^(.+?):(\d+):\d+: error: (.+?) \[([a-z-]+)\]`
- 提取文件路径、行号、错误消息和错误码

### 修复逻辑
- 按文件分组处理错误
- 从后往前修复（避免行号变化）
- 检查已有 `# type: ignore` 注释并合并
- 保持代码格式不变

### 智能修复（type-arg）
尝试自动添加类型参数：
- `Dict` → `Dict[str, Any]`
- `List` → `List[Any]`
- `Set` → `Set[Any]`
- `dict` → `dict[str, Any]`
- `list` → `list[Any]`
- `set` → `set[Any]`
