# Archive Directory

This directory contains archived documentation, reports, and development notes that are no longer actively maintained but preserved for historical reference.

## Contents

- `admin-test-reports/` - Archived test reports from Django Admin testing phases
- `insurance-dev-notes/` - Archived development notes from insurance service implementation
- `CLEANUP_REPORT.md` - Project structure cleanup report (2024-12)
- `PERFORMANCE_TEST_RESULTS.md` - Archived performance test results

## Purpose

Files in this directory are:
- Historical records of development progress
- Reference material for understanding past decisions
- Not actively maintained or updated

## Note

If you need to reference these files for current development, consider whether the information should be updated and moved to the appropriate active documentation directory.
