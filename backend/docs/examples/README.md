## 🔍 如何使用示例代码

### 1. 查看示例

每个示例都包含：
- **完整的代码实现**
- **详细的注释说明**
- **使用方法和参数说明**
- **运行示例的方法**

### 2. 运行示例

大部分示例可以直接在Django shell中运行：

```bash
# 进入Django shell
python manage.py shell

# 复制示例代码并运行
```

### 3. 集成到项目

示例代码可以：
- 直接复制到项目中使用
- 作为开发新功能的参考模板
- 用于理解功能的工作原理
