## ✨ 主要功能

- **客户与当事人中台（Client）**
  - 客户基础档案、证件信息、财产线索统一管理
  - 支持从自然语言解析客户信息并结构化入库
  - 支持客户-案件-合同三方联动查看
- **合同全生命周期（Contracts）**
  - 合同创建、补充协议、定稿材料、律师分配
  - 付款记录、催收提醒、合同台账查询
  - 支持合同审查任务管理与结果回看
- **案件中台（Case）**
  - 案件创建、编辑、状态推进、进度追踪
  - 案号管理、案件日志、案件当事人、律师指派
  - 案件访问授权与协作留痕，适配多人并行办案
- **文书与内容生产（Documents + Litigation AI）**
  - 诉讼文书模板化生成与结构化填充
  - AI 诉讼文书生成、证据清单组织、外部模板映射
  - 合同审查任务、文书生成任务全流程可追溯
- **证据与材料管理（Evidence）**
  - 证据文件归档、整理、排序与展示
  - 支持案件材料组织、导出与复用
  - 支持聊天记录梳理、录屏抽帧与导出任务
- **法律检索与类案匹配（Legal Research）**
  - 二阶段匹配：先宽召回（关键词/BM25/向量/段落信号混合），再精排（LLM 结构化判定）
  - 双模型复核：边界分数样本自动触发复判，降低高分误判
  - 查询扩展：同义词扩展 + LLM 检索式变体 + 伪相关反馈扩展检索
  - 动态语义策略：词法优先，低置信样本再启用语义向量，平衡命中率与耗时
  - 可配置最大扫描数、最低相似度、LLM 模型；支持任务进度追踪、取消、候选池提示与 PDF 归档下载
  - API 优先 + Playwright 回退，私有 API 缺失时自动降级，保证通用可用
- **自动化执行与外部系统联动（Automation + OA）**
  - 法院短信解析与文书下载、法院网站自动抓取
  - OA 立案联动、立案流程引导（Onboarding）
  - 财产保全保险询价、飞书群通知、流程自动推进
- **金融与专项计算（Finance + Sales Dispute + Reminders）**
  - 利息/违约金/LPR 相关计算能力
  - 买卖纠纷等专项业务计算工具
  - 关键日期提醒、催收提醒与结果明细展示
  - 提醒模块已完成服务边界化（读写通过 ReminderService 统一对外）
- **组织与权限体系（Organization）**
  - 律师、团队、律所、账号凭证统一管理
  - 支持跨团队协作与权限边界控制
  - 已完成 organization 边界收敛：跨模块调用统一走 service/protocol，敏感字符串隐藏与 admin 密码遮罩行为保持不回归
- **平台基础能力（Core）**
  - 统一配置、异常体系、缓存、限流、健康检查
  - 统一日志与可观测能力，保障生产可运维性
- **开放平台能力（MCP Server）**
  - 支持 OpenClaw、Claude Desktop 等 Agent 直接调用系统能力
  - 提供案件、客户、合同、财务、组织、OA 立案等工具接口
