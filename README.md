## ✨ 主要功能（概览版）

如果你想快速理解这个系统做什么，看这一节就够了。  
更细的能力点与每次迭代细节，请直接看 [更新日志](CHANGELOG.md)。

- **客户与当事人（Client）**：客户档案、证件、财产线索统一管理，支持自然语言解析与企业信息回填。
- **合同管理（Contracts）**：合同全生命周期管理，覆盖分配、付款、催收、定稿材料与自动捕获。
- **案件管理（Case）**：案件进度推进、案号与日志管理、当事人/律师协作、案件材料扫描与归档。
- **文书生产（Documents + Litigation AI）**：模板化文书生成 + AI 辅助，支持执行事项规则引擎与裁判文书解析。
- **自动化与 OA（Automation + OA）**：法院短信解析下载与归档、飞书通知、财产保全询价、一张网立案联动。
- **法律检索（Legal Research）**：宽召回 + 精排复核的类案检索，支持 API 能力输出与检索轨迹解释。
- **PDF 拆解（PDF Splitting）**：支持内容识别拆分/按页拆分、OCR 档位选择、异步任务与 ZIP 下载。
- **企业数据（Enterprise Data）**：企业查询能力与 MCP 调试工作台，支持多 Key 与配置治理。
- **金融与提醒（Finance + Reminders）**：利息/LPR 计算、专项争议计算、关键日期与催收提醒。
- **组织权限（Organization）**：律师、团队、律所、账号权限统一管理，支持多人协作与边界控制。
- **平台基础（Core）**：统一配置、异常、缓存、限流、日志与异步任务能力。
- **开放接口（MCP Server）**：支持 Agent 直接调用案件、客户、合同、财务、组织、OA 等能力。
